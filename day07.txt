변수라는 값으로 데이터를 저장하려면 메모리 안에서
연속적으로 값을 삭제하고 다시 추가해야한다.
자료형 변수를 지정하여서 여러 데이터를 담을수 있는 구조를 자료구조

1). 배열을 선언하는 방법
1.자료형 [] 변수 이름 ; = int [] arr
2. 자료형 변수 이름 [];= int arr [] ;

*자료형 뒤에 붙이거나 변수명 뒤에 붙이면 
자료형 배열이다. 가독성을 높이는 방법으로 선언
-기본자료형
int long = 0
float double =0.0
-참조 자료형
자료형 [] 변수 이름 = null; // 공간이 있다 없다 !

2) 배열의 선언과 생성
배열을 선언한것은 배열을 다루는 참조변수 공간만
만들어진것이다. 저장할 수있는 공간을 만들려면
생성해야된다
*배열을 선언하는 방법
int [배열 연산자] arr =new int [배열크기];

*배열 선언하고 생성할떄 오류 
int [] arr; // 배열변수선언
arr = {1,2,3,4,5}; // 오류발생
*배열 선언하고 재정의 방법
int [] arr;
arr = new int[] {1,2,3,4,5}; //배열의 재정의 o

3) 배열 메모리 구조
int [  ] arr = new int [5];->//참조 변수 : 주소를 출력하면 배열의 위치가 주소값으로 출력된다.

일반변수 : 데이터를 직접 갖고오는 변수
(short, byte , int , double)
참조변수 : 데이터 위치가 주소값을 가지는 변수로 해당 주소를 접근해서 갖고온다.
(String / 배열 / collections / 객체 (object)

@배열의 구조
1)인덱스 : 배열의 각 공간의 위치값을 말한다.
배열이 지나다니는 위치를 인덱스(index) 라고
말하며 공간을 숫자로 표현한다.
0부터 시작하는 시작의 숫자를 의미하며
공값의 값을 가져오는 의미는 공간의 첨자 [0]=인덱스를 
부여해서 사용한다.

2)배열의 길이 : 배열의 변수가 할당된 크기를 함수 배열의 길이를 함수로 사용한다.
int arr_size = arr.length;
System.out.println(arr_size);
System.out.println(arr.length); 

3)배열의 초기값 : 선언을 했을때 자동으로 지정되는 초기값이다
정수형 :0 /실수형 : 0.0 / 문자형 : '' /객체형 :Null

@Array(배열 출력 /배열 정렬 / 배열 복사)

배열을 간단하게 활용하는 방법을 도와주는 클래스 
-배열의 복사 (얕은복사/깉은복사)
1.얕은 복사 (shallow copy)
복사된 배열이나 원본 배열이 변경 될때 서로 간의 값이 변경된다
int [] arr01 = {1,2,3};
int [] arr02 = arr01;

2.깉은 복사 (deep copy)
복사된 배열이나 원본 배열이 변경될떄 서로 간의 값은 바뀌지 않는다
-복사할 배열을 선언하는 방법
int [] card = {1,6,4,5,3,2};
int [] newcard = new int [ card,length];
-Arrays.copyof(배열의 이름 ,배열길이)
*전체의 배열을 그대로 복사한형태
-System 함수 사용하기
System.arrayCopy(원본,0,복사본,0,배열크기)
*부분 복사를 깊은형태로 새로운 배열을 생성해서 만들수있다!
(원본지장없음/크기를 사용자정의)
